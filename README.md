# Digital Tech & Science News

Portal web estilo revista digital que consume la API de NewsAPI.org para mostrar las últimas noticias de tecnología y ciencia. Incluye tarjetas Bootstrap con diseño oscuro elegante y opción para descargar un archivo TXT con el Top 10 de noticias redactadas profesionalmente.

## Características
- Integración con NewsAPI.org
- Diseño moderno con paleta azul/gris/turquesa
- Tarjetas interactivas con imágenes y enlaces
- Botón para descargar Top 10 en TXT
- Footer fijo y estilo inspirado en portales de noticias
